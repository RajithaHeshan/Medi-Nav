<h1>Clinic Flow – iOS Mobile Application</h1>

<p>
A clean and user-friendly iOS app designed to improve the clinic and hospital visiting experience 
for patients, guardians, and medical staff.
</p>

<h2>📌 About the App</h2>

<p>
Clinic Flow helps users navigate clinics easily by providing features such as doctor channeling, 
real-time queue updates, clinic sequence guidance, indoor navigation, digital prescriptions, and 
emergency assistance. The app is built using <strong>SwiftUI</strong> with a modern medical UI theme.
</p>

<h2>🚀 Key Features</h2>

<ul>
  <li><strong>Doctor Channeling</strong> – Book appointments and view doctor availability</li>
  <li><strong>Clinic Sequence Display</strong> – Shows the full visit flow (Registration → Consultation → Pharmacy → Payment)</li>
  <li><strong>Real-Time Queue Status</strong> – Track your appointment number and doctor delays</li>
  <li><strong>Indoor Navigation</strong> – Search departments, rooms, and counters using map or voice</li>
  <li><strong>My Care Section</strong> – View medical reports, lab results, vitals, and prescriptions</li>
  <li><strong>Prescription Sharing</strong> – Send prescriptions to external pharmacies</li>
  <li><strong>Emergency Assistance</strong> – Request ambulance or wheelchair support</li>
  <li><strong>Notifications</strong> – Receive updates like bookings, reports, and reminders</li>
  <li><strong>Accessibility Support</strong> – Simple UI suitable for elderly and first-time visitors</li>
</ul>

<h2>🛠️ Technologies Used</h2>

<ul>
  <li>SwiftUI</li>
  <li>Xcode</li>
</ul>
